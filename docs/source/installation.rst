##############
 Installation
##############

The most recent release can be installed from `PyPI
<https://pypi.org/project/zenodo_client>`_ with uv:

.. code-block:: console

    $ uv pip install zenodo_client

or with pip:

.. code-block:: console

    $ python3 -m pip install zenodo_client

*********************
 Installing from git
*********************

The most recent code and data can be installed directly from GitHub with uv:

.. code-block:: console

    $ uv pip install git+https://github.com/cthoyt/zenodo-client.git

or with pip:

.. code-block:: console

    $ python3 -m pip install git+https://github.com/cthoyt/zenodo-client.git

****************************
 Installing for development
****************************

To install in development mode with uv:

.. code-block:: console

    $ git clone git+https://github.com/cthoyt/zenodo-client.git
    $ cd zenodo-client
    $ uv pip install -e .

or with pip:

.. code-block:: console

    $ python3 -m pip install -e .
